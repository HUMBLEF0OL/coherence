## What is Coherence
Coherence is a Claude Code plugin that detects documentation drift — when code changes without updating the docs that describe it — and proposes surgical, reviewable patches. It runs as Claude Code hooks and slash commands, with a two-stage LLM pipeline that plans, then writes, patches.
